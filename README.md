# random_number · AstrBot 插件

发送 `/rn`,返回一个 **0 到 6 之间的随机整数**(含 0 和 6)。

管理员(`1412219758`)可用 `/rnrig` 做一个**全方位骰子控制台**:定向受害者、固定数值、限定范围、屏蔽数字、概率偏置、次数限制……

## 命令

| 命令          | 别名             | 说明                          |
| ------------- | ---------------- | ----------------------------- |
| `/rn`         | `随机数`、`roll` | 随机返回 0-6 的整数           |
| `/rnrig`      | —                | 管理员骰子控制台(仅指定 QQ) |

## `/rnrig` 控制台(仅管理员 `1412219758`)

### 确定性控制(优先级从高到低)

| 用法                                          | 说明                                    |
| --------------------------------------------- | --------------------------------------- |
| `/rnrig <0-6>`                                | 下一掷(任何人)固定为该数,**一次性**  |
| `/rnrig keep <0-6>`                           | 持续固定为该数,直到关闭                |
| `/rnrig keep <0-6> times <N>`                 | 持续固定 N 次后**自动恢复**随机         |
| `/rnrig target <qq> <0-6>`                    | 指定 qq 的下一掷固定,**一次性**        |
| `/rnrig target <qq> keep <0-6> [times <N>]`   | 针对 qq 持续(可限定次数)             |
| `/rnrig target <qq> off`                      | 取消对该 qq 的所有定向作弊              |

### 随机约束(未被确定性控制命中时才生效)

| 用法                       | 说明                                   |
| -------------------------- | -------------------------------------- |
| `/rnrig range <min> <max>` | 随机只落在 `[min, max]`                |
| `/rnrig norange`           | 清除范围限制                           |
| `/rnrig block <n> [n...]`  | 永远不出这些数                         |
| `/rnrig unblock <n> [n...]`| 解除屏蔽                               |
| `/rnrig noblock`           | 清除所有屏蔽                           |
| `/rnrig bias <n> <1-100>`  | 该数有指定百分比概率出现               |
| `/rnrig nobias`            | 清除概率偏置                           |

### 管理

| 用法            | 说明               |
| --------------- | ------------------ |
| `/rnrig status` | 查看当前全部状态   |
| `/rnrig off`    | 一键清空所有控制   |
| `/rnrig help`   | 显示帮助           |

## 优先级与玩法示例

掷骰解析顺序:**定向一次性 → 全局一次性 → 定向持续 → 全局持续 → 约束随机**。

```
# 1. 只坑一个人一次
/rnrig target 123456 0          →  123456 下一次必出 0,之后恢复正常

# 2. 群里全员倒霉,但你自己没事
/rnrig keep 0                   →  所有人都出 0
/rnrig target 1412219758 keep 6 →  只有管理员一直出 6

# 3. 隐蔽控场:只出 4-6,看着像运气好
/rnrig range 4 6

# 4. 保底:永不出 0
/rnrig block 0

# 5. 6 出现 50% 概率
/rnrig bias 6 50

# 6. 限时整蛊:接下来 5 掷全是 6,之后自动恢复
/rnrig keep 6 times 5

# 查看当前所有设置
/rnrig status

# 收摊
/rnrig off
```

## 安装

将本文件夹放入 AstrBot 的 `data/plugins/` 目录下,然后在 AstrBot 管理面板中**重载插件**即可。

> QQ 平台无需额外配置 —— AstrBot 会自动把 QQ 收到的 `/rn`、`/rnrig` 消息路由到本插件。

## 修改管理员

编辑 `main.py` 顶部的常量:

```python
ADMIN_QQ = "1412219758"   # ← 改成你自己的 QQ
```

## 备注

- 所有控制状态保存在**内存**中,AstrBot 重启 / 重载插件后会清空(对整人骰子来说更安全,避免忘了关 `keep 6`)。
- 非管理员使用 `/rnrig` 会返回"⛔ 此功能仅限管理员使用。"。
