# random_number · AstrBot 插件

发送 `/rn`,返回一个 **0 到 6 之间的随机整数**(含 0 和 6)。

## 命令

| 命令     | 别名             | 说明                  |
| -------- | ---------------- | --------------------- |
| `/rn`    | `随机数`、`roll` | 随机返回 0-6 的整数   |

> 别名通过 `@filter.command("rn", alias={"随机数", "roll"})` 注册。

## 安装

将本文件夹放入 AstrBot 的 `data/plugins/` 目录下,然后在 AstrBot 管理面板中**重载插件**即可。

> QQ 平台无需额外配置 —— AstrBot 会自动把 QQ 收到的 `/rn` 消息路由到本插件。

## 示例

```
用户:/rn
机器人:🎲 你抽到了:3
```
