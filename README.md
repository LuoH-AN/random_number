# random_number · AstrBot 插件

发送 `/rn`,返回一个 **0 到 6 之间的随机整数**(含 0 和 6)。

管理员(`1412219758`)可用 `/rnrig` 作弊控制结果。

## 命令

| 命令          | 别名             | 说明                          |
| ------------- | ---------------- | ----------------------------- |
| `/rn`         | `随机数`、`roll` | 随机返回 0-6 的整数           |
| `/rnrig`      | —                | 管理员作弊骰子(仅指定 QQ)   |

### `/rnrig` 子命令(仅管理员 `1412219758`)

| 用法                 | 说明                                       |
| -------------------- | ------------------------------------------ |
| `/rnrig <0-6>`       | **下一掷**固定为该数,用一次后自动恢复随机 |
| `/rnrig keep <0-6>`  | **持续**固定为该数,直到关闭               |
| `/rnrig off`         | 取消所有控制,恢复随机                     |
| `/rnrig status`      | 查看当前作弊状态                          |

非管理员使用 `/rnrig` 会返回"⛔ 此功能仅限管理员使用。"。

## 安装

将本文件夹放入 AstrBot 的 `data/plugins/` 目录下,然后在 AstrBot 管理面板中**重载插件**即可。

> QQ 平台无需额外配置 —— AstrBot 会自动把 QQ 收到的 `/rn` 消息路由到本插件。

## 示例

```
用户:/rn
机器人:🎲 你抽到了:3

管理员:/rnrig keep 6
机器人:🎯 已持续作弊:接下来每一掷都是 6。

用户:/rn
机器人:🎲 你抽到了:6
```
